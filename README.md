# CI491 Project

## Installation guide:

Please install Python 3 to run this script.

**Install Pip:**

curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

python3 get-pip.py

**Install Beautiful Soup:**

`pip install beautifulsoup4`

**Install Google:**

`pip install google`

**Install Spacy:**

`pip install spacy`

`python -m spacy download en_core_web_sm`
